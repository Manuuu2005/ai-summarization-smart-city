# AI Summarization for Smart City Compliance

## Description
This project is a desktop-based AI application that summarizes smart city compliance documents using NLP.

## Features
- Upload text files
- AI-based summarization using LexRank
- Simple GUI interface

## Technologies Used
- Python
- Tkinter
- NLTK
- Sumy

## How to Run
pip install -r requirements.txt
python main.py
